About ID Ai

Created by Isara Dilnuka

ID AI is a simulated Ai search engine built by Isara Dilnuka, a passionate developer from Horana, Sri Lanka. This project is designed to showcase the look and feel of a smart Ai-driven search interface.

Although this version does not connect to a real Ai backend, it simulates how an Ai system might understand and respond to user inputs. Future versions aim to integrate real Ai models for more advanced capabilities.

Vision & Purpose

ID Ai was created for learning and demonstration purposes. It reflects Isara’s interest in AI, web development, and human-computer interaction. As a student developer and designer, Isara focuses on building modern web apps that are clean, interactive, and mobile-responsive.

Features

  • ✓ Simulated Ai responses based on structured query templates
  • ✓ Basic context-aware follow-up interactions (predefined)
  • ✓ Fully responsive design powered by Tailwind CSS
  • ✓ Clean, modern UI designed by Isara
  • ✓ Expandable project structure for future integration with real AI APIs
